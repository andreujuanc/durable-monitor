# durable-monitor
Azure durable functions cli monitor

## Installation

### npx
```bash
npx @andreujuanc/durable-monitor --help
```
### global install
```bash
npm install -g @andreujuanc/durable-monitor
```

## Usage
```bash
andreujuanc/durable-monitor --help
```

or use the alias
```bash
durable-monitor --help
```
